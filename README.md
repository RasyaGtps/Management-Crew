## ğŸ“¦ Instalasi

1. **Clone Repository**
   ```bash
   git clone https://github.com/rasyagtps/management-crew.git
   cd management-crew
   ```

2. **Install Dependencies**
   ```bash
   composer install
   npm install
   ```

3. **Setup Environment**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Konfigurasi Database**
   - Buat database baru
   - Update konfigurasi database di file `.env`
   ```env
   DB_DATABASE=management_crew
   DB_USERNAME=root
   DB_PASSWORD=
   ```

5. **Migrasi Database**
   ```bash
   php artisan migrate
   ```

6. **Compile Assets**
   ```bash
   npm run dev
   ```

7. **Jalankan Server**
   ```bash
   php artisan serve
   ```
